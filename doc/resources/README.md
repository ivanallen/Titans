## 重要说明

### 1. 关于代码风格

- 逗号后面留空格
- 双目运算符两边留空格
- 左花括号不换行
- if/while/for/switch 等关键字后面留空格
- 所有文件、文件夹命名请使用小写下划线
- 所有源码内的使命名，请使用小写下划线
- 缩进使用 4 空格，禁止使用 TAB。如果你使用 IDE，请把 TAB 配置成 4 空格。如果你使用 vim，请使用我提供的 vim 配置，链接在这里：https://gitee.com/ivan_allen/myvim
