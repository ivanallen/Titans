### 3. 你能理解`open`函数`flags`标志位`O_APPEND`是如何实现的吗？
答：`O_APPEND`表示追加。如果文件已有内容，这次打开文件所写的数据附加到文件末尾而不覆盖原来的内容。  
    因为`write`函数调用执行后文件指针总是在文件尾部。
